## 💻 Começando

**Clone o repositório do projeto**

```bash
$ git clone https://github.com/GustavoCunhaLacerda/huggy-store.git && cd huggy-store
```

**Siga os passos**

```bash
# Instalando as dependências
$ npm i

# Certifique de possuit um .env válido

# Iniciar a aplicação
$ npm run dev
```
